# Asfandiar — Reader Feedback

This is the public feedback repository for *Asfandiar*, a memoir-novel
based on the life of Dr. Asfandiar Shukri.

If you found this repository, you were probably sent here from the
book's reading site (https://io-kai.github.io/asfandiar/) by a link
in the in-page "Feedback" form. Submitting that form opens a new
issue here, pre-filled with the chapter you were reading and the note
you wrote.

## How to use this

1. Read a chapter on the site.
2. Click the **Feedback** button in the bottom-right corner.
3. Write your correction, memory, or note. (Include photographs by
   dragging them into the GitHub issue page after the form opens.)
4. Press **Send**. GitHub will open the pre-filled issue in a new tab.
5. If you don't have a GitHub account yet, follow the prompts on
   GitHub to sign up. It is free and takes about a minute. You can
   sign up with the email address you already use.
6. Click **Submit new issue**. The author will read it and respond.

No collaborator invitation is required. Anyone with a GitHub account
can post here.

## Privacy

Issues posted here are publicly visible. If you have something
private to share, email it to brian@remoteimagingconsultants.com
instead.

The novel manuscript itself is in a separate private repository.
This repository is *only* for receiving feedback.
